dialog

An awesome lightweight jQuery plugin for creating overlay dialogs.

features

  • Show overlay dialogs
  • Open and stack multiple dialogs on top of already open dialogs
  • Custom heading and controls for dialogs
  • Close dialogs on press of 'Esc' key or on blur

installation

Installation is super easy. Just follow these steps:

  1. Add jQuery library to your page.

<script type="text/javascript" src="js/jquery.js"></script>

  1. Add the dialog script file to your page.

<script type="text/javascript" src="js/dialog.js"></script>

  1. Add the dialog css file to your page.

<link rel="stylesheet" type="text/css" href="css/dialog.css" />

That's it.!! Easy right ? Now, go ahead and use the plugin to create awesome dialog boxes.

how to create dialogs using this plugin

Nothing special needs to be done. Using this plugin, you can convert your existing HTML content into awesome dialog boxes. Here's how to do it.

Lets say we have the following DIV with ID 'myFirstDialog' on our page which we want to convert into a dialog box. 

<div id="myFirstDialog">
    <div class="dialog-content">
        My first basic dialog box.!!
    </div>
</div>

Now, to turn this DIV into a dialog box, initialize it with the dialog plugin with the following code: 

<script>
    $(document).ready(function() {
        $('#myFirstDialog').dialog();
    });
</script>

You can also pass various options while initializing the dialog. To see a list of all the available options, read the documentation of the plugin here. Once initialized, you can show and hide the dialog by triggering the 'dialog-open' and 'dialog-close' events respectively on the DIV like this: 

<script>
    function showMyFirstDialog() {
        $('#myFirstDialog').trigger('dialog-open');
    }

    function hideMyFirstDialog() {
        $('#myFirstDialog').trigger('dialog-close');
    }
</script>

To add a 'Title' to the dialog, add the following code to your DIV: 

<div id="myFirstDialog">
    <div class="dialog-title">Title for the Dialog box</div>

    <div class="dialog-content">
        My first basic dialog box.!!
    </div>
</div>

To add an 'Actions bar' to the dialog, add the following code to your DIV: 

<div id="myFirstDialog">
    <div class="dialog-title">Title for the Dialog box</div>

    <div class="dialog-content">
        My first basic dialog box.!!
    </div>

    <div class="dialog-controls">
        <button type="button">Close this Dialog</button>
    </div>
</div>

And that's it. As simple as that. I hope you enjoy using the plugin. :)
